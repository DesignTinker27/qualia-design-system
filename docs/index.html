Installation & Usage

Get started with the Qualia Design System in your project quickly and efficiently.

📦 Install Package

npm install @qualia/design-system

🎨 Import Design Tokens

You can import tokens as JavaScript modules or include the CSS custom properties file.

// JavaScript/React\nimport { colors, spacing, typography } from '@qualia/design-system/tokens';\n\n// CSS (e.g., in your main CSS file or a PostCSS setup)\n@import '@qualia/design-system/tokens/tokens.css';

🔘 Use Components

Import and use React components directly in your application. Ensure you've imported the main CSS bundle for component styling.

import { Button } from '@qualia/design-system';\n\nfunction MyApp() {\n  return (\n    <Button \n      variant="primary" \n      size="large"\n      onClick={handleClick}\n    >\n      Get Started\n    </Button>\n  );\n}

🔄 Migration from Semantic UI

Transitioning from Semantic UI? Our components offer direct mappings to streamline the migration process.

// Before (Semantic UI)\n<Button primary size="large" loading={loading}>\n  Submit\n</Button>\n\n// After (Qualia Design System)\n<Button variant="primary" size="large" loading={loading}>\n  Submit\n</Button>

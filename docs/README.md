# Qualia Design System Documentation

Welcome to the Qualia Design System documentation site. This directory contains all documentation resources for designers, developers, and stakeholders.

## 📁 Structure

- **index.html** - Main documentation site with interactive demos
- **tokens/** - Design token documentation and examples
- **components/** - Component-specific usage guides
- **guidelines/** - Design and development guidelines
- **examples/** - Real-world usage examples

## 🚀 Quick Start

1. **View the docs**: Open `index.html` in your browser
2. **Host locally**: Use any static server (e.g., `python -m http.server`)
3. **Deploy**: Push to GitHub Pages for live documentation

## 📱 Features

- **Interactive demos** - Live component examples
- **Token visualization** - Color palettes, spacing scales
- **Usage guidelines** - Best practices and patterns
- **Installation guides** - Step-by-step setup
- **Migration help** - Semantic UI → Qualia transition
- **Responsive design** - Mobile-friendly documentation

Button Component

The primary button component for all Qualia products. Supports multiple variants, sizes, and states with built-in accessibility.

React Component

<QualiaButton variant="primary" size="medium" showIcon={false} disabled={false} aria-label="Example button for testing interactions"> Click me </QualiaButton>

HTML

<button class="qualia-button qualia-button--primary qualia-button--medium" aria-label="Example button for testing interactions"> Click me </button>

CSS Tokens Used

/* Primary Button Tokens */ background-color: var(--interactive-01); /* #1a874e */ color: var(--text-04); /* #ffffff */ border-radius: var(--radius-xs); /* 4px */ padding: var(--space-08) var(--space-16); /* 8px 16px */ min-height: 40px; /* Medium size */

Built-in accessibility features:

  • Keyboard navigation support
  • Focus visible indicators
  • Screen reader labels
  • ARIA attributes
  • Color contrast compliance
